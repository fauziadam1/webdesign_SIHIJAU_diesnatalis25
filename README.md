# Bank Sampah – Trash2Cash ID

Website untuk pengelolaan bank sampah dan edukasi lingkungan yang ramah pengguna. Fitur utama:
- Beranda dengan hero, cara kerja, manfaat, jenis sampah, tips edukasi, dan lokasi.
- Autentikasi sederhana (login/daftar) dan profil pengguna.
- Dashboard admin: statistik, transaksi, anggota, jenis sampah, serta grafik interaktif.
- Teknologi: Next.js App Router, Tailwind CSS v4, ikon lucide-react, dan chart recharts.

## Persyaratan
- Node.js v18+ dan npm
- Koneksi internet untuk mengunduh dependensi

## Instalasi
bash
npm install


## Menjalankan di Lokal
bash
npm run dev

- Buka http://localhost:3000
- Halaman default mengarah ke /user (beranda pengguna)
- Halaman Profile User: /user/profile
- Area admin demo: /admin/dashboard

## Build & Production
bash
npm run build
npm start


## Daftar Skrip 
- npm run dev – mode pengembangan
- npm run build – build produksi
- npm start – jalankan hasil build
- npm run lint – linting kode
